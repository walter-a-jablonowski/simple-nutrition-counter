Shared use
----------------------------------------------------------

### Sources

- single string (url or just text)
- or array

```yaml

sources:

  - title:           # all entries (non-required)
    source:          # e.g. url
    sub:             # sub section
    comment:         # details

  1:                 # or named index for referencing

  { myType: , ... }  # or like in foods
```
